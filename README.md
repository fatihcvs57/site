# DesignPack Pro

Basit bir başlangıç iskeleti. Sunucu Express + TypeScript, istemci React + Vite + Tailwind.

## Çalıştırma

Projeyi geliştirmek için gerekli bağımlılıkları yükleyin:

```bash
npm install --prefix server
npm install --prefix client
```

Sunucuyu geliştirme modunda çalıştırmak için:

```bash
npm run dev --prefix server
```

İstemciyi başlatmak için:

```bash
npm run dev --prefix client
```

Her iki servis de hazır olduğunda tarayıcıdan `http://localhost:5173` adresine giderek uygulamayı görüntüleyebilirsiniz.
